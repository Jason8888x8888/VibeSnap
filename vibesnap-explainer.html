PART 01

技术架构：一个切开的“三层蛋糕”

我们的项目其实就像一个精心烘焙的蛋糕，每一层都有自己独特的味道和分工。

🍰

顶层糖霜 (面子) / 前端界面 (React + Tailwind)

大白话： 这是你直接通过眼睛看到、用鼠标点到的漂亮网页。它负责接收你上传的图片，也负责把你想要的颜色、排版漂漂亮亮地展示出来。它就像餐厅里的“服务员兼领位员”，只负责服务你，不懂做菜。

🤖

中层夹心 (大脑) / AI 引擎 (Google Gemini)

大白话： 这里的隐形大佬。当“顶层糖霜”把你的图片送进来时，这个极其聪明的“AI 厨神”会一眼看穿图片的灵魂——什么颜色、什么字体、怎么排版。它把这些玄学感觉翻译成精确的数据字典，然后扔回给顶层。

☁️

底层烤盘与菜谱 (地基) / 基础设施 (Vercel & GitHub)

大白话： 这个蛋糕得装在盘子里大家才能吃，对吧？
Vercel： 就是那个24小时不打烊的“云端展柜”，它把我们的网站放到网上，让全世界任何人只要输入网址就能访问。
GitHub： 就是我们存放“菜谱配方（祖传代码）”的保险箱，每次我们改进了配方，都会存在这里。Vercel 只要看到配方更新，就会立刻烘焙一个新蛋糕摆上展柜！

我们的 SaaS 工具大揭秘！ (到底要花多少钱？)

一个人就像一家公司，我们这次一个人也没招，全部“白嫖”了全球顶尖的云服务替我们打工。这就是现代开发者的“工具库”。

GitHub (祖传藏宝洞)

它的角色是？

天下第一的代码大仓库。我们所有的项目文件、甚至我的工作记忆，都存在这。就像一个拥有无数把锁的安全金库。

免费给了啥？

永远不要钱。 免费无限建仓库，免费放任意多代码，连私人小项目也不收钱。

什么时候才要掏钱？

只有当你开了一家大公司（比如几十个人的开发团队），想要高级的管理权限和企业级安全时，才需要交“保护费”（团队版 $4/人/月 起）。

Vercel (最猛的包租婆)

它的角色是？

负责把你写在本地的网页拔起来，扔到公网上去让所有人都能通过域名访问。顺带，它跟 GitHub 穿一条裤子，你在 GitHub 改个字，它立刻重新发布。

免费给了啥？

对个人超级大方！ 免费送二级域名（就是咱们那个冗长的网址），每个月免费给你跑 100GB 的流量（对于个人玩具来说根本跑不完），全自动防外部攻击。

什么时候才要掏钱？

你的产品火了，变成了一个日活好几千人的正式商业项目，或者你需要拉上你的小伙伴联合协作时，升级到 Pro 版（$20/月 起）。

Gemini (免费白嫖的超算)

它的角色是？

Google 最能打的 AI 模型。我们调用的专门用于接外部程序的 API 接口。是它赋予了这个项目看到图片、并把“高级感”翻译成人话的核心魔法！

免费给了啥？

目前极其离谱的白嫖额度！ 对于 `Gemini 1.5 Flash`，免费每天让你提问 1500次（每分钟最多15次）！这对一个人的实验项目来说，简直是拿着高射炮打蚊子。

什么时候才要掏钱？

当你项目上线，每天有几十上百人疯狂上传图片解析，触碰到了每分钟 15 次的限电闸门导致报错时（我们之前就经历过降级）。这时你需要绑定海外信用卡转成按次付费形式（其实依然很便宜，几美分可以调一千次）。
